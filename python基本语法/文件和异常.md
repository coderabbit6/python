## 从文件中读取数据

#### 读取整个文件
- 创建一个txt文件
- 读取文件
```
with open('pai.txt') as f:
	con = f.read()
	print(con)
```
- 可在print语句中使用rstrip()消除输出的空格

#### 文件路径
- 提供文件路径它让Python到系统的特定位置去查找
- 相对路径.\,\,
- 绝对路径file_path = 'C:\Users\ehmatthes\other_files\text_files\filename.txt'

#### 逐行读取
```
with open('pai.txt') as f:
	for line in f:
		print(line)
```
